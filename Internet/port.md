# PORT

**설명**

> 한 번에 둘 이상에 연결을 해야하는 상황일 때 서로 구분을 해주기 위해 사용한다.

> TCP 패킷정보를 보면 출발지 PORT, 도착지 PORT가 있다.

> ip는 목적지 서버를 찾는것, 서버 안에서 돌아가는 어플을 찾는것이 PORT이다.

**비유**

> ip는 한 아파트, port는 몇동몇호라고 비유할 수 있다.

- 0 ~ 65535 할당 가능
- 0 ~ 1023 : 잘 알려진 포트, 사용하지 않는 것이 좋음
  - FTP - 20,21
  - TELNET - 23
  - HTTP - 80
  - HTTPS - 443
